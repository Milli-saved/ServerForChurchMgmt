# What are the necessary database tables in this system?

## Users (leaders )

-> Full Name
-> password
-> phone number
-> email
-> Responsible for what team

## Members

-> Full Name
-> phone number
-> email
-> in what team
-> address
-> emergency contact name and phone number
-> participating in what programs

## Teams (yeagelgelot kifloch)

-> Team name
-> Team leader
-> Team

## Programs

-> Program Name
-> Program day and time
-> Program organizer
